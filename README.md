# 36-校园反诈系统(小程序)

[文档地址](http://wechat.zjrcsy.cn/)

### 技术栈和环境:

基于 B/S 架构 SpringBoot+MySQL+Vue+elementui+用户端Uniapp
环境:  Idea mysql maven jdk1.8

### 功能:

#### 用户

1.首页: 展示反诈知识、法律法规、反诈活动、新闻资讯等的信息  可进行评论或报名参与

2.诈骗测评: 可进行知识测评

3.论坛交流: 可进行发帖  对帖子可进行回复 点赞和收藏

4.我的:  

​         个人信息设置

​          有防诈智能客服进行咨询对话

​          查看已活动报名

​          查看举报信息

​          查看我的收藏

​          查看诈骗测评记录

​          查看错题本

​          查看我的发帖

​          可在线留言

​          推荐反诈知识功能





#### 管理

1.首页:展示echarts统计分析图表,包括学生总数、反诈知识汇总数、活动报名汇总数、举报信息汇总数

2.学生管理: 增删改查

3.知识类型以及反诈知识: 增删改查

4.法规类型以及法律法规: 增删改查

5.活动类型以及反诈活动: 增删改查

6.活动报名: 审核及删改查

7.举报类型及举报信息: 增删改查

8.论坛分类: 增删改查

9.敏感词: 对敏感词进行查看和修改

10.举报记录: 进行查看、删除和举报审核

11.诈骗测评管理:  进行增删改查

12.试题管理: 增删改查(填空题、判断题、多选题、主观题、单选题)

13.在线留言查看及回复  删除

14.论坛交流: 查看 删除 及查看评论

15.系统管理: 轮播图、新闻资讯、新闻资讯分类

16.诈骗测评记录和错题本查看

17.个人中心: 个人信息修改和查看
